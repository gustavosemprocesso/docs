---
title: "Autenticação"
description: "Como se autenticar para enviar requisições para a nossa API"
---

Para autenticar com nossa API você precisa de uma chave de API: É com essa chave que nossa API identifica a sua conta e te permite enviar requisições com sucesso.

## Obter uma chave de API

Para obter uma chave de API será necessário entrar em contato com o nosso suporte através do e-mail suporte@semprocesso.com.br

## Utilizar a chave de API

Para garantir a sua autenticação durante o uso da API, será preciso acrescentar o header `x-api-key` nas requisições. Caso o header não seja informado ou seja inválido a nossa API retornará o código HTTP `401`.
