## Chave de API

Primeiramente, é necessário que você obtenha uma chave de API através do nosso suporte e posteriormente acrescente esta chave como um header nas requisições realizadas.

Exemplo de utilização do header HTTP:

```json
"headers": {
  "x-api-key": "<YOUR_API_KEY>"
}
```

Feito isso, você está pronto para começar a fazer requisições e obter os seus dados através da API do Lexxy.
